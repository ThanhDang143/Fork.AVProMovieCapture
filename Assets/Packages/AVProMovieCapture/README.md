# Fork.AVProMovieCapture
Powerful high-performance yet easy to use audio-video capture solution for Unity.

## Installation
### Unity Package Manager
```
https://github.com/ThanhDang143/Fork.AVProMovieCapture.git?path=/Assets/Packages/AVProMovieCapture
```

1. In Unity, open **Window** → **Package Manager**.
2. Press the **+** button, choose "**Add package from git URL...**"
3. Enter url above and press **Add**.

### Scoped Registry

1. In Unity, open **Project Settings** → **Package Manager** → **Add New Scoped Registry**
- ``Name`` ThanhDVs
- ``URL`` https://upm.thanhdv.com
- ``Scope(s)`` thanhdv

2. In Unity, open **Window** → **Package Manager**.
- Press the **+** button, choose "**Add package by name...**" → ``thanhdv.renderheads.avpro-movie-capture``
- or
- Press the **Packages** button, choose "**My Registries**"
